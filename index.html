    Arabic Alphabet Vanilla JS

    This is my very first project which I used to learn what it actually means to write your own code or apply code from tutorials to your own project. I decided for my first project I will make an interactive book which teaches you the Arabic Alphabet, as this looked appealing to me in the sense that it looked like it was something easy to design and I can focus more on learning Javascript fundamentals.

    Features
    • One page Application
    • Ajax request pages therefore no need for multiple html files
    • Fixed Bottom Navigation
    • Jump to specific pages using select options
    • Listen to individual letter sounds by clicking on the letter
    • Loop through the the all letters and sounds
    • Bookmark pages
    Steps
    1. Learned JavaScript Fundamentals and DOM manipulations.
    2. Used Adobe Photoshop to design letters and IPhone 5 to record sounds.
    3. Designed the first page as a start with Bootstrap
    4. Studied Ajax crash course to use xhttp request. I didnâ€™t want to have a lot of HTML pages.
    5. Studied JSON crash course to store the Name of the sounds and image names.
    6. I took the code form a gallery slider to help navigate requests between pages by incrementing/decrementing with index of the array from the JSON.
    7. Learned to apply select option to navigate to specific pages thus requesting the specified index number in the array which provided the paths to the Images.
    8. Added sound functionality
    9. Learned local storage to add bookmark
    Problems and Solutions
    • I was trying to figure out how to get the property of an array and was trying to get the values of object. After some google searching I found out about javascripts built in Object object which had a property of values which was a simple soultion to store in a variable
